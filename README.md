# 基于bootstrap和seajs后台模板
```
一套基于bootstrap和seajs的后台模板，一键式的js配置文件，不需要在写html
页面模板格式基本固定，如果要有特定的开发，需自行进行二次开发
这是基于特定情况开发的，只有参考意义
其中存在的一些bug只能自行解决了，毕竟个人是没打算在去维护
```

# 注意
```
1、本套模板只是个人查阅资料，根据公司需求开发，是嵌套在java里面使用的
2、在本地浏览有跨域的问题，需使用webstorm或vsCode（配置debug模式）打开。（因用的是ajax请求本地json文件，跨域问题自行百度）
3、seajs的引用在本地打开可能会出现路径问题，请自行查看源码解决，因为这套基本不会去更新了，用的技术栈有点过时了。只是里面用的技术还是有参考价值。
4、有个uEditor编辑器已被注释，因本地引用路径问题。图片上传也有些问题，含有公司上传图片的特有逻辑
```

# 文件目录
```
json    本地json文件请求
module  模块js
pages   页面html
public  公共的静态文件
script  js文件
template_h5+  h5+的html模板页面，这是整套基于bootstrap开发的html模板，网上搜罗的。
```
